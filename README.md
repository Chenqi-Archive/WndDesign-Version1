# WndDesign图形界面库

An effective and friendly c++ Win32 gui library based on Direct2D and DirectWrite.

纯c++编写的Win32图形界面库，底层依赖于Direct2D和DirectWrite。



使用Visual Studio 2019，c++17编译。



一些使用样例，请参考[样例代码](./example.md)。
